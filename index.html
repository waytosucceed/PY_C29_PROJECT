Save The Spaceship-II

In the previous class’s project, you set up the foundation of Save the Spaceship by displaying the spaceship and introducing meteors and stars as space objects. But so far, they’ve just been sitting in one place. Now, it’s time to bring motion into the game!

In this project, you’ll make meteors and stars fall from the top of the screen, just like real space debris and collectibles. Instead of appearing in fixed positions, each object will spawn at a random spot and move downward at different speeds, adding a sense of unpredictability. You’ll also ensure that new objects appear at controlled intervals rather than all at once, making the game more balanced.

Here are the things you need to do:

Step 1: Open Your Project

  • Open the folder where you saved your "Save the Spaceship" game in the last class.
  • Open main.py in your code editor.

Step 2: Modify the SpaceObject Class

  • Update the __init__() method to:
    • Set self.x to random.randint(0, WIDTH - 100)
    • Set self.y to -50
    • Set self.speed_y to random.randint(3, 7)
  • Add a move() method to move objects downward by adding self.speed_y to self.y

Step 3: Create a List to Store Objects
Before the game loop, create an empty list named objects to store both meteors and stars.

Step 4: Add a Timer for Spawning Objects

  • Define a variable spawn_interval = 2000 to control how often objects appear.
  • Define last_spawn_time = 0 to track when the last object was created.

Step 5: Check for New Objects to Spawn

  • Inside the game loop, store pygame.time.ticks() in current_time.
  • Create an if block to check at least 2 seconds have passed since the last object spawn or if last_spawn_time is 0 and if either of it is true do the following:
    • Create a new meteor or star and add it to objects list.
    • Update last_spawn_time to current_time.

Step 6: Remove Off-Screen Objects
Use list comprehension to remove objects that have moved below the screen.

Step 7: Draw and Move Objects
Loop through objects, call draw() to display them, and move() to update their position.

Step 8: Run the Game
Save your file and run main.py to see meteors and stars falling continuously while the spaceship remains in place.
